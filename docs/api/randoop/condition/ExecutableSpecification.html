randoop.condition

Class ExecutableSpecification

  • java.lang.Object
    • randoop.condition.ExecutableSpecification

  • public class ExecutableSpecification
extends Object
    The executable version of an OperationSpecification. It allows the underlying Boolean expressions to be evaluated.

    It is represented as the collection of all ExecutableBooleanExpression, GuardPropertyPair, and GuardThrowsPair for the Precondition, Postcondition, and ThrowsCondition objects defined on a single operation. Includes specifications inherited from supertypes.

      • checkPrestate

        private void checkPrestate(Object[] args,
                           ExpectedOutcomeTable table)
        Modifies the given table, adding an ExpectedOutcomeTable entry for the guard expressions of this method's local specification recording the following:
        1. Whether the preExpressions fail or are satisfied. See checkPreExpressions(java.lang.Object[]).
        2. A set of ThrowsClause objects for expected exceptions. See checkGuardThrowsPairs(java.lang.Object[]).
        3. The expected ExecutableBooleanExpression, if any. See checkGuardPropertyPairs(java.lang.Object[]).
        Parameters:
        args - the argument values
        table - the table to which the created entry is to be added
