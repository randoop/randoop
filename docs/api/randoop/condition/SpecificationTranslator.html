      • getExpressionSignature

        private static RawSignature getExpressionSignature(AccessibleObject executable,
                                                   boolean postState)
        Create the RawSignature for the expression method for evaluating an expression for the given method or constructor.

        The parameter types of the RawSignature are the declaring class as the receiver type, followed by the parameter types of executable, followed by the return type if postState is true and executable is not a void method.

        Note: The declaring class of the expression method is actually determined by ExecutableBooleanExpression.createMethod(RawSignature, String, String, SequenceCompiler)

        Parameters:
        executable - the method or constructor to which the expression belongs
        postState - if true, include a variable for the return value in the signature
        Returns:
        the RawSignature for a expression method of executable
