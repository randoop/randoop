randoop.generation

Class AbstractGenerator

  • java.lang.Object
    • randoop.generation.AbstractGenerator
  • Direct Known Subclasses:
    ForwardGenerator

    public abstract class AbstractGenerator
extends java.lang.Object
    Algorithm template for implementing a test generator.

    The main generation loop is defined in method explore(), which repeatedly generates a new sequence, determines if it a failing sequence, and stops the process when the time or sequence limit expires. The process of generating a new sequences is left abstract.

      • stop

        protected boolean stop()
        Tests stopping criteria and determines whether generation should stop. Criteria are checked in this order:
        • if there is a listener manager, RandoopListenerManager.stopGeneration() returns true,
        • the elapsed generation time is greater than or equal to the max time in milliseconds,
        • the number of output sequences is equal to the maximum output,
        • the number of generated sequences is equal to the maximum generated sequence count, or
        • if there is a stopper, IStopper.stop() returns true.
        Returns:
        true if any of stopping criteria are met, otherwise false
