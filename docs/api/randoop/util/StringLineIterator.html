      • nextLine

        public java.lang.String nextLine(int length)
        Returns a line of words, ensuring that the length of the line is no greater than the given length. If such a line is not possible, returns an empty String.

        Note that if length is always too small for the iterator to make progress by returning non-empty lines, the iterator will not terminate.

        Parameters:
        length - the maximum line length
        Returns:
        the next line of words, no longer than maximum length
