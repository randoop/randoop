Class Randomness

  • java.lang.Object
    • randoop.util.Randomness

  • public final class Randomness
extends Object
    A simple-to-use wrapper around Random.

    It also supports logging, for debugging of apparently nondeterministic behavior.

      • binarySearchForIndex

        private static int binarySearchForIndex(double[] cumulativeWeights,
                                        double point)
        Performs a binary search on a cumulative weight distribution and returns the corresponding index i such that cumulativeWeights.get(i) < point <= cumulativeWeights.get(i + 1) for 0 <= i < cumulativeWeights.length.
        Parameters:
        cumulativeWeights - the cumulative weight distribution to search through. The ith element is the cumulative weight of all elements before the ith (that is, exclusive rather than inclusive). The last (i+1)th element is the weight of all elements.
        point - the value used to find the index within the cumulative weight distribution
        Returns:
        the index corresponding to point's location in the cumulative weight distribution
