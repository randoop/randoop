Class ReflectionCode

  • java.lang.Object
    • randoop.util.ReflectionCode
  • Direct Known Subclasses:
    ConstructorReflectionCode, MethodReflectionCode, ObjectContractReflectionCode

    public abstract class ReflectionCode
extends Object
    Wraps a method or constructor together with its arguments. Can be run only once. hasRun() indicates whether it has been run.

    Implemented by parts of Randoop that want to execute reflection code via ReflectionExecutor.

      • runReflectionCode

        public final void runReflectionCode()
                             throws ReflectionCode.ReflectionCodeException
        Runs the reflection code that this object represents.
        1. If System.getSecurityManager() returns a RandoopSecurityManager, this method sets the security manager's status to ON.
        2. This method calls runReflectionCodeRaw() to perform the actual work. runReflectionCodeRaw() sets the .retVal or exceptionThrown field, or throws an exception if there is a bug in Randoop.
        3. This method sets the security manager's status to its status before this call.
        Throws:
        ReflectionCode.ReflectionCodeException - if execution results in conflicting error and success states; this results from a bug in Randoop

      • runReflectionCodeRaw
