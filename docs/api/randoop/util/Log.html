      • log

        private static void log(String s)
        Log to GenInputsAbstract.log, if that is non-null.

        This is private because it is better to use logPrintf(java.lang.String, java.lang.Object...) than to call log with a concatenation. In other words: 

        
 log("arg1=" + arg1);      // BAD
 logPrintf("arg1=%s", arg1); // GOOD
        The reason is that if arg1.toString() fails, #logPrintf can catch that exception, but the exception would occur before logLine is entered.
        Parameters:
        s - the string to output

      • logLine

        private static void logLine(String s)
        Log to GenInputsAbstract.log, if that is non-null.

        This is private because it is better to use logPrintf(java.lang.String, java.lang.Object...) than to call logLine with a concatenation. In other words: 

        
 logLine("arg1=" + arg1);      // BAD
 logPrintf("arg1=%s%n", arg1); // GOOD
        The reason is that if arg1.toString() fails, #logPrintf can catch that exception, but the exception would occur before logLine is entered.
        Parameters:
        s - the string to output (followed by a newline)
