Class ExpectedExceptionGenerator

  • java.lang.Object
    • randoop.test.ExpectedExceptionGenerator
  • All Implemented Interfaces:
    TestCheckGenerator

    public class ExpectedExceptionGenerator
extends java.lang.Object
implements TestCheckGenerator
    A TestCheckGenerator that generates checks for exceptions that are expected at a particular statement. Creates a ExpectedExceptionCheck that is returned in a RegressionChecks collection if the exception occurs, or a ErrorRevealingChecks collection if not.

    Note that this generator is distinct from other check generators that either return regression checks or error-revealing checks.
