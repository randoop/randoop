Interface ExceptionPredicate

  • All Known Implementing Classes:
    AlwaysFalseExceptionPredicate, ExceptionBehaviorPredicate

    public interface ExceptionPredicate
    An ExceptionPredicate is a test on an exception primarily used to check whether the exception is classified as expected. Used in TestCheckGenerator implementations to decide whether an exception matches criteria of that check generator.
