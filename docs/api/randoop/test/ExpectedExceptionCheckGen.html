randoop.test

Class ExpectedExceptionCheckGen

  • java.lang.Object
    • randoop.test.ExpectedExceptionCheckGen

  • public class ExpectedExceptionCheckGen
extends java.lang.Object
    A check generator that based on an ExceptionPredicate when given an exception, generates an ExpectedExceptionCheck object when the predicate is satisfied, and an EmptyExceptionCheck otherwise. Resulting tests only enforce expected matching exceptions.
    See Also:
    RegressionCaptureVisitor.visit(randoop.sequence.ExecutableSequence)
