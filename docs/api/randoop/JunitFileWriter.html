Class JunitFileWriter

  • java.lang.Object
    • randoop.JunitFileWriter

  • public class JunitFileWriter
extends java.lang.Object
    JunitFileWriter is a class that for a collection of sequences, outputs Java files containing one JUnit4 test method per sequence. An object manages the information for a suite of tests (name, package, and directory) and is used by first running writeJUnitTestFiles and then writeSuiteFile or writeDriverFile.
