    public class CloneVisitor
extends Object
implements com.github.javaparser.ast.visitor.GenericVisitor<com.github.javaparser.ast.Node,Object>
    CloneVisitor is a visitor which makes a deep copy of a JavaParser AST node. Unlike the CloneVisitor provided by JavaParser 2.4.0, this class also copies each node's orphan comments.
