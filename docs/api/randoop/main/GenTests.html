      • createTestCheckGenerator

        public TestCheckGenerator createTestCheckGenerator(VisibilityPredicate visibility,
                                                   ContractSet contracts,
                                                   MultiMap<Type,TypedOperation> observerMap,
                                                   java.util.Set<TypedOperation> excludeAsObservers)
        Creates the test check generator for this run based on the command-line arguments. The goal of the generator is to produce all appropriate checks for each sequence it is applied to. Validity and contract checks are always needed to determine which sequences have invalid or error behaviors, even if only regression tests are desired. So, this generator will always be built. If in addition regression tests are to be generated, then the regression checks generator is added.
        Parameters:
        visibility - the visibility predicate
        contracts - the contract checks
        observerMap - the map from types to observer methods
        excludeAsObservers - methods to exclude when generating observer map
        Returns:
        the TestCheckGenerator that reflects command line arguments.
