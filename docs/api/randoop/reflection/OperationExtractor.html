randoop.reflection

Class OperationExtractor

  • java.lang.Object
    • randoop.reflection.DefaultClassVisitor
      • randoop.reflection.OperationExtractor
  • All Implemented Interfaces:
    ClassVisitor

    public class OperationExtractor
extends DefaultClassVisitor
    OperationExtractor is a ClassVisitor that creates a collection of Operation objects for a particular ClassOrInterfaceType through its visit methods as called by ReflectionManager.apply(Class). Allows types of operations of an InstantiatedType to be instantiated using the substitution of the type.
    See Also:
    ReflectionManager, ClassVisitor
