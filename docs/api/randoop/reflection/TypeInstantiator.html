      • instantiateSortedSetType

        private Substitution<ReferenceType> instantiateSortedSetType(TypedClassOperation operation)
        Returns a substitution that instantiates the SortedSet type of the given constructor. A SortedSet may be built so that the element type E is ordered either explicitly with a Comparator<E>, or the element type satisfies E implements Comparable<E>. The SortedSet documentation indicates that there should be four constructors, each of which indicates the order to be used and so constrains the element type.
        Parameters:
        operation - the constructor for the SortedSet subtype to be instantiated
        Returns:
        the substitution to instantiate the element type of the SortedSet type

      • instantiateDeclaringClass

        private Substitution<ReferenceType> instantiateDeclaringClass(ClassOrInterfaceType declaringType)
        Chooses an instantiating substitution for the given declaring class. Performs a coin flip to determine whether to use a previous instantiation, or to create a new one. Either way, if a new instantiating substitution is required, uses selectSubstitution(List, Substitution) to construct and choose one. Verifies that all of the type parameters of the declaring type are instantiated, and logs failure if not.
        Parameters:
        declaringType - the type to be instantiated
        Returns:
        a substitution instantiating the given type; null if none is found

      • selectMatch

        private Substitution<ReferenceType> selectMatch(ClassOrInterfaceType declaringType,
                                                ClassOrInterfaceType patternType)
        Selects an existing type that instantiates the given pattern type which is an instantiation of a generic declaring type and returns the instantiating substitution for the more general type. The pattern type makes it possible to select matches for partial instantiations of a generic type.

        Note: for all uses to work properly, the input types need to be closed on supertypes: if a type is in the input types, then so are all of its supertypes.

        Parameters:
        declaringType - the generic type for which instantiation is to be found
        patternType - the generic type from which match is to be determined, must be instantiation of declaringType.
        Returns:
        a substitution instantiating given type as an existing type; null if no such type

      • selectSubstitution

        private Substitution<ReferenceType> selectSubstitution(List<TypeVariable> typeParameters,
                                                       Substitution<ReferenceType> substitution)
        Extends the given substitution by instantiations for the given list of type variables. If any of the type variables has a generic bound, assumes there are dependencies and enumerates all possible substitutions and tests them. Otherwise, independently selects an instantiating type for each variable.
        Parameters:
        typeParameters - the type variables to be instantiated
        substitution - the substitution to extend
        Returns:
        the substitution extended by instantiating type variables; null if a variable has no instantiating types

      • selectAndExtend

        private Substitution<ReferenceType> selectAndExtend(List<TypeVariable> parameters,
                                                    Substitution<ReferenceType> substitution)
        Selects types independently for a list of type parameters, and extends the given substitution by the substitution of the selected types for the parameters.

        IMPORTANT: Should only be used for parameters that have non-generic bounds.

        Parameters:
        parameters - a list of independent type parameters
        substitution - the substitution to extend
        Returns:
        the substitution extended by mapping given parameters to selected types; null, if there are no candidate types for any parameter
