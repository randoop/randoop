randoop.contract

Class PrimValue

  • java.lang.Object
    • randoop.contract.ObjectContract
      • randoop.contract.PrimValue

  • public final class PrimValue
extends ObjectContract
    A check recording the value of a primitive value obtained during execution, (e.g. var3 == 1 where var3 is an integer-valued variable in a Randoop test).

    Obviously, this is not a property that must hold of all objects in a test. Randoop creates an instance of this contract when, during execution of a sequence, it determines that the above property holds. The property thus represents a regression as it captures the behavior of the code when it is executed.
