      • buildConditionMethodName

        private static java.lang.String buildConditionMethodName(Tag tag,
                                                         int tagIndex,
                                                         int methodIndex)
        Constructs the condition method name using the convention with Toradocu. Uses the index of the corresponding DocumentedMethod in the JSON list, and the index of the tag in the method tag set flattened to a list. The name is the letter 'm' followed by the method index, an underscore, and then the tag type string followed by the tag index. So, for instance, the method name for the first throws tag of the first method will be "m0_t0", while the name for the 2nd param tag of the 5th method would be "m4_p1".
        Parameters:
        tag - the tag for the condition, must be non-null
        tagIndex - the position of the tag of the tag list
        methodIndex - the position of the method in the JSON list
        Returns:
        the constructed method name
