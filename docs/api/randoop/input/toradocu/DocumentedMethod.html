randoop.input.toradocu

Class DocumentedMethod

  • java.lang.Object
    • randoop.input.toradocu.DocumentedMethod

  • public final class DocumentedMethod
extends java.lang.Object
    DocumentedMethod represents the Javadoc documentation for a method in a class. It identifies the method itself and key Javadoc information associated with it, such as throws tags, parameters, and return type.

    * Modified class from Toradocu used to deserialize the JSON output of Toradocu.
