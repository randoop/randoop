Class FailingTestFilter

  • java.lang.Object
    • randoop.output.FailingTestFilter
  • All Implemented Interfaces:
    CodeWriter

    public class FailingTestFilter
extends Object
implements CodeWriter
    A CodeWriter that outputs JUnit tests with assertions that fail commented out. Intended to be used with regression tests in order to filter flaky tests that pass within Randoop, but fail when run from the command line.

    Writes the class, and then compiles and runs the tests to determine whether there are failing assertions. Each failing assertion is replaced by a comment containing the code for the failing assertion. Creates a clean temporary directory for each compilation/run of a test class to avoid state effects due to files in the working directory.
