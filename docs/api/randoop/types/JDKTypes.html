randoop.types

Class JDKTypes

  • java.lang.Object
    • randoop.types.JDKTypes

  • public class JDKTypes
extends Object
    Defines type constants for classes in the JDK Collections.

    These types should be maintained as new JDK versions are released. Updated for JDK 8.
