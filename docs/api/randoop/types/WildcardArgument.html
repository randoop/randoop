Class WildcardArgument

  • java.lang.Object
    • randoop.types.TypeArgument
      • randoop.types.WildcardArgument
  • Direct Known Subclasses:
    WildcardArgumentWithLowerBound, WildcardArgumentWithUpperBound

    abstract class WildcardArgument
extends TypeArgument
    Represents a wildcard type argument to a parameterized type. A wildcard may have either an upper or lower bound as defined in JLS Section 4.5.1. 
       ? [ extends ReferenceType ]
   ? [ super ReferenceType ]

    The subclasses represent the type bound as given for the wildcard.
