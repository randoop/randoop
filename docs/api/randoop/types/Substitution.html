randoop.types

Class Substitution<T>

  • java.lang.Object
    • randoop.types.Substitution<T>

  • public class Substitution<T>
extends Object
    Manages the substitution of concrete types for type variables and wildcards in an instantiation of a generic class as a parameterized type.

    Because a substitution represents the instantiation from a generic class to a parameterized type, an instance is built using forArgs(List, List) and then not modified.
