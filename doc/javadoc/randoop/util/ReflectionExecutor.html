public final class ReflectionExecutor
extends Object

Executes the code of a ReflectionCode object. This class maintains an "executor" thread. Code is executed on that thread. If the code takes longer than the specified timeout, the thread is killed and a ReflectionExecutor.TimeoutExceeded exception is reported.
